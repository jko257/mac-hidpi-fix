## 先说清楚：为什么 Mac 外接显示器会模糊

**HiDPI（Retina 渲染）**是 macOS 让屏幕显示更细腻的核心技术。简单说，它用 4 个物理像素渲染 1 个逻辑像素，让字体边缘平滑、图标锐利，就是你在 MacBook 内置屏幕上看到的那种效果。

问题来了：苹果只对 4K 及以上分辨率的外接显示器自动开启 HiDPI。你接了一台 2K（2560×1440）显示器，系统默认不开，字体就会有毛边、发虚。这不是显示器质量问题，是 macOS 的限制。

以前解决这个问题需要关闭系统完整性保护（SIP）、手动改配置文件，折腾半天还不一定成功。BetterDisplay 把这个流程变成了几次点击。

## 用 BetterDisplay 解决外接显示器模糊：操作步骤

这是最常见的需求，专业版的核心场景之一。操作不复杂，全程不需要关闭 SIP。

**准备工作：** 先下载 BetterDisplay，完成安装，购买专业版并激活。

**开启 HiDPI 自定义分辨率的步骤：**

1. 点击菜单栏 BetterDisplay 图标，选择**设置（齿轮图标）**
2. 在设置页面找到你的外接显示器，点击进入显示器详情
3. 找到 **「Edit the default system configuration of this display」**，勾选
4. 找到 **「Add custom scaled resolution」**，勾选
5. 输入目标 HiDPI 分辨率，比如 2K 屏推荐输入 `1920×1080` 或 `2048×1152`，点击右侧加号添加
6. 点击 **「Apply Change」**，重启 Mac
7. 重启后进入「系统设置 → 显示器」，选择你刚才设定的分辨率，字体清晰度立刻不同

重启后这个分辨率设置会保留在系统里，不需要每次开机都运行 BetterDisplay 才生效。
